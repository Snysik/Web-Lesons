Основні властивості css для роботи з текстом

Властивість font-size

Властивість font-size змінює розмір шрифта.

Розмір шрифта задається за допомогою значення та одиниць виміру (em,px,pt,%)

Синтаксис:

CSS 

        p{
            font-size: 20px;
        }

HTML

<p>Hello, world!<p>

Результат:

Hello, world!

Властивість font-weight

Властивість font-weight встановлює насиченість шрифту.


Може приймати наступні значення:

  • normal - нормальна насиченість, те саме, що і 400
  • bold - напівжирний текст, те саме що і 700
  • lighter - змінює насиченість відносно батьківського стилю в меншу сторону
  • bolder - змінює насиченість відносно батьківського стилю в більшу сторону
  • 100, 200, 300, 400, 500, 600, 700, 800, 900 - цифрові значення насиченості, які відрізняються від стандартних

Синтаксис:

CSS 

                    
                    .bold{
                    font-weight: bold;
                    }
                    .thin{
                    font-weight: 100;
                    }
                    .max{
                    font-weight: 900;
                    }

HTML

<p>
<span class = "bold">напівжирний текст</span>
<span class = "thin">найтонший текст</span>
<span class = "max">найтовщий текст</span>
<p>

Результат:

                

                     напівжирний текст ,
                     найтонший текст ,
                     найтовщий текст .

Властивість font-style

Властивість font-style визначає накреслення шрифту - звичайне, курсивне або похило.


Може приймати наступні значення:

  • normal - Звичайне накреслення тексту
  • italic - Курсивне накреслення
  • oblique - Похиле накреслення

Синтаксис:

CSS 

                    
                        .normal{
                            font-style: normal;
                        }
                        
                        .italic{
                            font-style: italic;
                        }
                        
                        .oblique{
                            font-style: oblique;
                        }

HTML

<p>
<span class = "normal"> Звичайне накреслення тексту </span>
<span class = "italic"> курсивне накреслення тексту </span>
<span class = "oblique"> Похиле накреслення тексту </span>
<p>

Результат:

              

                 Звичайне  накреслення тексту
                 Курсивне  накреслення тексту
                 Похиле  накреслення тексту

Властивість font-family

Властивість font-family визначає, який шрифт використати.


Може приймати наступні значення:

  • serif - шрифти із засічками (антиквенні), типу Times;
  • sans-serif - рубані шрифти (шрифти без засічок чи гротески), типовий представник - Arial;
  • monospace - моноширинні шрифти, ширина кожного символу у такому сімействі однакова (шрифт Courier).

Синтаксис:

CSS 

                    
                        .serif{
                            font-family: serif;
                        }
                        .sans-serif{
                            font-family: sans-serif;
                        }
                        .monospace{
                            font-family: monospace;
                        }

HTML

<p>
<span class = "serif"> шрифти із засічками (антиквенні), типу Times; </span>
<span class = "sans-serif"> рубані шрифти (шрифти без засічок чи гротески), типовий представник - Arial; </span>
<span class = "monospace"> моноширинні шрифти, ширина кожного символу у такому сімействі однакова (шрифт Courier). </span>
<p>

Результат:

            

                 шрифти із засічками (антиквенні), типу Times; 
                 убані шрифти (шрифти без засічок чи гротески), 
                типовий представник - Arial; 
                 моноширинні шрифти, ширина кожного символу у такому сімействі однакова 
                (шрифт Courier).

Властивість color

Властивість color змінює колір тексту


Може приймати наступні значення:

  • color - зміна коліру

Синтаксис:

CSS 

                            
                                .color{
                                    color: #3a7a43;
                                }

HTML

<p>
Зазвичай ліс <span class = "color"> зеленого </span> коліру
<p>

Результат:

Зазвичай ліс зеленого коліру

Властивість text-transform

Властивість text-transform Керує перетворенням тексту елемента в великі або великі символи


Може приймати наступні значення:

  • capitalize - Перший символ кожного слова в реченні буде великим. Інші символи свого вигляду не змінюють.
  • lowercase - Усі символи тексту стають малими (нижній регістр).
  • uppercase - Усі символи тексту стають великими (верхній регістр).
  • none - Не змінює регістр символів.

Синтаксис:

CSS 

                            
                                .capitalize{
                                    text-transform: capitalize;
                                }
                                .lowercase{
                                    text-transform: lowercase;
                                }
                                .uppercase{
                                    text-transform: uppercase;
                                }
                                .none{
                                    text-transform: none;
                                }

HTML

<p>
<span class = "capitalize"> перший символ кожного слова в реченні буде великим. </span> Інші символи свого вигляду не змінюють.
<span class = "lowercase"> Усі символи тексту стають малими (нижній регістр). </span>
<span class = "uppercase"> Усі символи тексту стають великими (верхній регістр). </span>
<span class = "none"> Не змінює регістр символів. </span>
<p>

Результат:

                

                    перший символ кожного слова в реченні буде великим.  
                    Інші символи свого вигляду не змінюють.
                    Усі символи тексту стають малими (нижній регістр). 
                    Усі символи тексту стають великими (верхній регістр). 
                    Не змінює регістр символів.

Властивість text-decoration

Властивість text-decoration Додає оформлення тексту як його підкреслення, перекреслення чи лінії над текстом.


Може приймати наступні значення:

  • blink - Встановлює миготливий текст. Такий текст періодично приблизно раз на секунду зникає, потім знову з'являється на колишньому місці.
  • line-through - Створює перекреслений текст
  • overline - Лінія проходить над текстом
  • underline - Встановлює підкреслений текст
  • none - Скасовує всі ефекти, у тому числі й підкреслення посилань, яке задано за замовчуванням.

Синтаксис:

CSS 

                        
                            .blink{
                                text-decoration: blink;
                            }
                            .line-through{
                                text-decoration: line-through;
                            }
                            .overline{
                                text-decoration: overline;
                            }
                            .underline{
                                text-decoration: underline;
                            }
                            .none2{
                                text-decoration: none;
                            }

HTML

<p>
Встановлює <span class = "blink">миготливий текст. </span> Такий текст періодично приблизно раз на секунду зникає, потім знову з'являється на колишньому місці.
Створює перекреслений <span class = "line-through">текст </span>
Лінія проходить <span class = "overline">над текстом </span>
Встановлює <span class = "underline">підкреслений текст </span>
<span class = "none2"> Скасовує всі ефекти, у тому числі й підкреслення посилань, яке задано за замовчуванням. </span>
<p>

Результат:

                

                    Встановлює  миготливий текст.  
                    Створює перекреслений  текст 
                    Лінія проходить  над текстом 
                    Встановлює  підкреслений текст 
                    Скасовує всі ефекти, у тому числі й підкреслення посилань, 
                    яке задано за замовчуванням
